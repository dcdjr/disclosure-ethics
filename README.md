# The Ethics of Cybersecurity Vulnerability Disclosure

This repository contains a research paper analyzing the ethical dilemmas surrounding cybersecurity vulnerability disclosure. The paper examines standardized definitions, disclosure models, and the competing responsibilities of researchers, vendors, and governments. A detailed case study of the Log4Shell (CVE-2021-44228) incident is used to show how disclosure processes can succeed or fail in real-world environments.
